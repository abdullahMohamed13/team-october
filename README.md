# Libraries Installed

## React Router
`npm install react-router-dom`

## Yup
`npm install yup`
<br />

## Redux
`npm install @reduxjs/toolkit react-redux`

## Tailwindcss
`npm install tailwindcss @tailwindcss/vite`
<br />
`npm install -D @types/node`
<br />

## Shadcn
`npx shadcn@latest init`
